▓▓ NETPYWIZ // NETWORK MONITOR ▓▓
──────────────────────────────────

INSTALL (recommended):
  sudo bash installer_linux.sh
  Then run: sudo NetPyWiz

RUN WITHOUT INSTALLING:
  sudo ./NetPyWiz

REQUIREMENTS:
  - Linux (Debian/Ubuntu/Mint)
  - Must run as sudo (needs raw socket access)
  - ieee-data package (installer handles this automatically)

USAGE:
  1. Enter your subnet in CIDR format e.g. 192.168.1.0/24
  2. Wait for scan to complete
  3. Monitor devices in real time
  4. Double-click SW PORT column to label patch cables
  5. Click END SESSION + EXPORT to save CSV to Desktop
